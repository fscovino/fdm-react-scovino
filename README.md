# FDM React Project

This is a React.js project which utilizes {JSON} Placeholder API to test CRUD methods on To-do and Users.

## Dependencies

- React JS
- Bootstrap 5

### Installing Liraries

- React.js

```
npm install react react-dom
```

- Bootstrap

```
npm install bootstrap
```

- JQuery

```
npm install jquery
```

- Popper

```
npm install @popperjs/core
```
